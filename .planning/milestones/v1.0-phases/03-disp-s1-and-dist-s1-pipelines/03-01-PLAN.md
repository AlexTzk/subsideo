**2. Create `src/subsideo/products/disp.py`** with these functions, following the thin-wrapper pattern from cslc.py (per D-01):

- `_validate_cds_credentials(cdsapirc_path: Path) -> None` -- per D-03, check file exists and contains `url:` and `key:` lines. Raise `FileNotFoundError` with message: `"CDS API config not found at {path}. ERA5 tropospheric correction is mandatory for DISP. Register at https://cds.climate.copernicus.eu/"`. If file exists but missing `url:` or `key:`, raise `ValueError("CDS API config at {path} is incomplete -- must contain url: and key: fields")`.

- `_run_dolphin_phase_linking(cslc_file_list: list[Path], work_dir: Path, coherence_threshold: float) -> tuple[list[Path], list[Path]]` -- per D-04, lazy import `from dolphin.workflows.config import DisplacementWorkflow` and `from dolphin.workflows.displacement import run as dolphin_run`. Create `DisplacementWorkflow(cslc_file_list=cslc_file_list)`. Call `dolphin_run(cfg)`. Return `(outputs.stitched_ifg_paths, outputs.stitched_cor_paths)`. Log with loguru: `"dolphin phase linking complete: {} interferograms"`.

- `_apply_coherence_mask(ifg_paths: list[Path], cor_paths: list[Path], threshold: float) -> list[Path]` -- per D-06, for each ifg/cor pair: load with rasterio, create a boolean mask where coherence < threshold, write masked interferograms to `{ifg_stem}_masked.tif`. Return list of masked ifg paths.

- `_run_unwrapping(masked_ifg_paths: list[Path], cor_paths: list[Path], work_dir: Path) -> list[Path]` -- lazy import tophu. For each interferogram: load ifg and cor arrays with rasterio, call `tophu.multiscale_unwrap(ifg=ifg_data, corr=cor_data, nlooks=1.0, unwrap_func=tophu.SnaphuUnwrap(cost="smooth", init="mcf"), downsample_factor=(3, 3), ntiles=(2, 2))`. Write unwrapped output to `{stem}_unwrapped.tif`. Return list of unwrapped paths.

- `_check_unwrap_quality(unwrapped_path: Path, threshold: float) -> dict` -- per D-05, load unwrapped phase with rasterio, build coordinate grid for valid (non-NaN) pixels, fit a plane via `scipy.linalg.lstsq` (A = [x, y, 1], b = phase values), compute residual RMS = `sqrt(mean((phase - plane)**2))`. Return `{"path": str(unwrapped_path), "residual_rms": float, "flagged": residual_rms > threshold}`. If flagged, log via loguru at WARNING level: `"Planar ramp anomaly detected in {path}: residual RMS {val:.3f} > threshold {threshold}"`.

- `_generate_mintpy_template(work_dir: Path, cdsapirc_path: Path) -> Path` -- create a MintPy smallbaselineApp.cfg template file at `work_dir / "smallbaselineApp.cfg"`. Key settings: `mintpy.load.processor = dolphin`, `mintpy.troposphericDelay.method = pyaps`, `mintpy.troposphericDelay.weatherModel = ERA5`, `mintpy.networkInversion.minTempCoh = 0.7`, `mintpy.timeFunc.polynomial = 1`. Return the cfg path.

- `_run_mintpy_timeseries(work_dir: Path, template_path: Path) -> list[Path]` -- per D-04, lazy import `from mintpy.smallbaselineApp import TimeSeriesAnalysis`. Create `TimeSeriesAnalysis(customTemplateFile=str(template_path), workDir=str(work_dir))`. Call `app.open()`, `app.run(steps=['load_data', 'modify_network', 'reference_point', 'invert_network', 'correct_troposphere', 'deramp', 'correct_dem_error', 'velocity'])`, `app.close()`. Collect HDF5 outputs: glob `work_dir / "*.h5"`. Return sorted list. Log: `"MintPy time-series inversion complete"`.

- `run_disp(cslc_paths: list[Path], output_dir: Path, cdsapirc_path: Path | None = None, coherence_mask_threshold: float = 0.3, ramp_threshold: float = 1.0) -> DISPResult` -- per D-01, the low-level orchestrator:
  1. Resolve cdsapirc_path (default `Path.home() / ".cdsapirc"`)
  2. `_validate_cds_credentials(cdsapirc_path)` -- fail fast per D-03
  3. `output_dir.mkdir(parents=True, exist_ok=True)`
  4. `_run_dolphin_phase_linking(cslc_paths, output_dir / "dolphin", coherence_mask_threshold)`
  5. `_apply_coherence_mask(ifg_paths, cor_paths, coherence_mask_threshold)`
  6. `_run_unwrapping(masked_ifgs, cor_paths, output_dir / "unwrap")`
  7. For each unwrapped path: `_check_unwrap_quality(path, ramp_threshold)` -- collect warnings
  8. `_generate_mintpy_template(output_dir / "mintpy", cdsapirc_path)`
  9. `_run_mintpy_timeseries(output_dir / "mintpy", template_path)`
  10. Find velocity file: `output_dir / "mintpy" / "velocity.h5"`
  11. Return `DISPResult(velocity_path=..., timeseries_paths=..., output_dir=output_dir, valid=True, qc_warnings=warnings)`
  Wrap the entire chain in try/except. On ImportError for any conda-forge dep, return `DISPResult(velocity_path=None, timeseries_paths=[], output_dir=output_dir, valid=False, validation_errors=[str])`.

- `run_disp_from_aoi(aoi: dict | Path, date_range: tuple[str, str], output_dir: Path, cdsapirc_path: Path | None = None, coherence_mask_threshold: float = 0.3, ramp_threshold: float = 1.0) -> DISPResult` -- per D-02, the end-to-end AOI+date range entry point:
  1. Resolve cdsapirc_path and validate CDS credentials immediately (fail fast before any CSLC processing).
  2. `aoi` is either a GeoJSON dict (with "type": "Polygon" and "coordinates") or a Path to a GeoJSON file. If Path, load with `json.load()`. Validate it has "type" and "coordinates" keys.
  3. Use the Phase 1 data layer to build the CSLC stack:

     b. Use CDSEClient to search for S1 IW SLC scenes covering the AOI within `date_range` (start, end): `from subsideo.data.cdse import CDSEClient`. Call `client.search(...)` and `client.download(...)` to get SAFE files.
     c. Fetch DEM for the AOI: `from subsideo.data.dem import fetch_dem`.
     d. For each scene, fetch orbit file: `from subsideo.data.orbits import fetch_orbit`.
  4. For each scene/burst combination, call `from subsideo.products.cslc import run_cslc` to produce CSLC HDF5 files. Collect all CSLC output paths into a list.
  5. Feed the CSLC HDF5 list into `run_disp(cslc_paths=cslc_list, output_dir=output_dir, ...)`.
  6. Return the DISPResult from run_disp().
  Wrap the CSLC-building loop in try/except. On failure, return `DISPResult(velocity_path=None, timeseries_paths=[], output_dir=output_dir, valid=False, validation_errors=[error_msg])`.
  Log progress at each stage: "Searching CDSE for {n} scenes...", "Building CSLC stack: {i}/{n}...", "CSLC stack complete, {n} files. Starting DISP pipeline...".

Use `from __future__ import annotations` at the top. Import `from pathlib import Path`, `from loguru import logger`, `from subsideo.products.types import DISPConfig, DISPResult`.
