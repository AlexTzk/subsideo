    from __future__ import annotations

    **Constants:**
    ```python
    JRC_BASE_URL = (
        "https://jeodpp.jrc.ec.europa.eu/ftp/jrc-opendata"
        "/GSWE/MonthlyHistory/LATEST/tiles"
    )
    # JRC grid: 30m resolution, 10-degree tiles = 40000 pixels
    JRC_TILE_SIZE_PIXELS = 40000
    JRC_TILE_SIZE_DEGREES = 10.0
    # Origin: upper-left at (-180, 80) (global coverage 80N to 60S)
    JRC_ORIGIN_LON = -180.0
    JRC_ORIGIN_LAT = 80.0
    ```

    **Internal functions:**

    1. `_jrc_tile_url(year: int, month: int, tile_x: int, tile_y: int) -> str` -- builds JRC Monthly History tile URL. tile_x and tile_y are indices (0-based). URL format: `{JRC_BASE_URL}/{year}/{year}_{month:02d}/{tile_x*40000:010d}-{tile_y*40000:010d}.tif`

    2. `_lonlat_to_jrc_tile(lon: float, lat: float) -> tuple[int, int]` -- converts geographic coordinate to JRC tile indices. tile_x = int((lon - JRC_ORIGIN_LON) / JRC_TILE_SIZE_DEGREES), tile_y = int((JRC_ORIGIN_LAT - lat) / JRC_TILE_SIZE_DEGREES).

    3. `_tiles_for_bounds(west: float, south: float, east: float, north: float) -> list[tuple[int, int]]` -- returns all JRC tile indices that cover a geographic bounding box.

    4. `_fetch_jrc_tile(url: str, cache_dir: Path) -> Path | None` -- downloads a JRC tile to cache_dir if not already cached. Use urllib.request.urlretrieve. Return None on 404 (some ocean tiles don't exist). Cache filename derived from URL.

    5. `_binarize_dswx(water_class: np.ndarray) -> np.ndarray` -- returns float32 array: classes 1,2 -> 1.0 (water), classes 0,3,4 -> 0.0 (not water), 255 -> np.nan. Per D-05: "DSWx water pixels" = high + moderate confidence.

    6. `_binarize_jrc(jrc: np.ndarray) -> np.ndarray` -- returns float32 array: value 2 -> 1.0 (water), value 1 -> 0.0 (not water), value 0 -> np.nan. Per Pitfall 5: JRC encoding is 0=nodata, 1=not water, 2=water.

    **Public function:**

    7. `compare_dswx(product_path: Path, year: int, month: int, cache_dir: Path | None = None) -> DSWxValidationResult` -- lazy import rasterio. Steps:
       a. Open product_path, get bounds and CRS
       b. Reproject bounds to EPSG:4326 to determine JRC tiles needed
       c. Download JRC tiles for the matching year/month (per D-05 temporal alignment)
       d. Mosaic JRC tiles covering the product extent
       e. Reproject product to JRC grid (30m WGS84) using rasterio.warp.reproject with Resampling.nearest
       f. Binarize both arrays
       g. Mask to valid pixels (both non-NaN)
       h. Compute f1_score, precision_score, recall_score, overall_accuracy
       i. Return DSWxValidationResult with pass_criteria={"f1_gt_0.90": f1 > 0.90}
